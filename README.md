# Data Cleaning Project: USA & Mexico Customer Base

## Project Overview
This project demonstrates professional data cleaning and formatting for a messy customer dataset. The goal was to prepare the data for seamless CRM import.

## What was fixed:
- **Deduplication:** Removed multiple redundant entries for the same customers.
- **Formatting:** Standardized names (Proper Case) and country labels.
- **Phone Validation:** Fixed scientific notation (E+11) and standardized international formats.
- **Visual Structure:** Applied corporate styling for better readability.

## Tools Used
- Microsoft Excel (Power Query, Formatting tools)
